# prompt ideas

## Prompt 1

Create a full desktop interface concept on a dark background, composed of floating biomorphic window modules with translucent plastic shells, glowing neon accents, embedded screens, rounded asymmetrical layouts, futuristic early 2000s inspired operating system, highly detailed, clean but expressive, no flat UI.

## Prompt 2

Design a mobile app interface system where all UI elements are soft blob shaped modules, glossy translucent materials, glowing buttons, radial layout clusters, futuristic playful OS design, black background, high detail, touch friendly shapes, liquid plastic aesthetic.

## Prompt 3

Generate a set of UI components including buttons, sliders, panels, and icons as isolated objects, each with translucent colored plastic, chrome accents, soft glow, rounded biomorphic forms, product style rendering on black background, consistent design language.

## Prompt 4

Create a futuristic operating system dashboard with modular floating control panels, equalizer like sliders, embedded waveform displays, glowing indicators, cybernetic interface feel, translucent shells, vibrant accent colors, highly detailed concept art, cinematic lighting.
