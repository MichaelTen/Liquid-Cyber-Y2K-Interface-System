# Liquid Cyber Y2K Interface System

A full interface design language for building entire applications, operating systems, and UI ecosystems inspired by liquid plastic futurism, early digital optimism, and expressive object-based interfaces.

This is not just about media players.

This is about rethinking how interfaces feel across:

- desktop systems
- mobile environments
- creative tools
- dashboards
- full UI frameworks

## Core idea

Interfaces should feel like objects.

Not flat layers.

Not invisible systems.

But tactile, glowing, modular structures that users interact with as if they are real.

## What this repository provides

- skill.md  
  A complete aesthetic specification for system-wide UI design

- prompt language  
  For generating assets and UI components

- visual rules  
  Shape, color, material, and composition guidance

- foundation for expansion  
  UI kits, system themes, launcher designs, and concept OS frameworks

## Where this can be used

- Linux custom desktop environments
- Windows shell replacements
- macOS inspired overlays
- Android launchers and skins
- Web-based operating systems
- Creative applications
- Experimental UI frameworks

## Design keywords

liquid plastic, translucent shell, glowing interface, blob geometry, embedded screens, cybernetic UI, object-based interface, retro futurist computing, luminous controls

## Philosophy

Modern interfaces became invisible.

This brings them back.

Not as noise, but as identity.

Not as clutter, but as expression.
